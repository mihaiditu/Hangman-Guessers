# Hangman-Guessers
Scripts for guessing the words from the Skeleton Hangman variants from dedge.com
# How to use
Compile and run the guess.cpp file for the respective variant. Type '-' in place of each letter you have to guess.
For the standard variant (Skeleton Hangman), type the category name before the hyphens (e.g. "food ------").
